# Plant Leaf Disease Detection (End-to-End ML Project)

Classification de 38 maladies sur 14 cultures à partir d'images de feuilles.

## Structure
- dataset/ → télécharger depuis Kaggle
- notebook/Plant_Disease_Detection.ipynb → EDA + entraînement
- model/ → modèle final sauvegardé
- api/ → Flask API
- web/ → interface web
